 ApniBooks - iOS Application

ApniBooks is an iOS application developed using **SwiftUI** that allows users to explore books through a clean and modern interface.  
This project was created as part of my learning journey in iOS development.

---

 Features

- Splash Screen
- User Authentication (Login & Sign Up)
- Home Screen with book categories
- Search screen with trending books
- Book detail view
- User profile screen
- Clean UI built with SwiftUI
- MVVM Architecture

--- Tech Stack

- Swift
- SwiftUI
- MVVM Architecture
- Xcode
- REST API Integration
